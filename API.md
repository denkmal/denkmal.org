## Get messages
Request:
```
GET /api/messages HTTP/1.1
Host: www.denkmal.org
```

Response:
```json
[
   {
      "id":4,
      "venue":2,
      "created":1371383458,
      "text":"Foo 1"
   },
   {
      "id":5,
      "venue":3,
      "created":1371383458,
      "text":"Foo 2"
   }
]
```


## Send message
Request:
```
POST /api/message HTTP/1.1
Host: www.denkmal.org
content-type: text/plain
content-length: 65

venue=1&text=foobar&hash=72ec3a3a9a0723aee528d20fb84a81e554ec4c71
```
Response:
```json
{
	"id":13,
	"venue":1,
	"created":1380379451,
	"text":"foobar"
}
```
